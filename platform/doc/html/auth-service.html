Authentication Service - API

Process SignIN - method GET

Ο client στέλνει ένα GET request στον authentication server, με υποχρεωτική την παράμετρο appID. Εάν ο authentication server δεν εντοπίσει την συγκεκριμένη παράμετρο, επιστρέφει μήνυμα λάθους και τερματίζει την διαδικασία. Εφόσον ο authentication server εντοπίσει την παράμετρο appID, ψάχνει στον εσωτερικό πίνακα αντιστοίχισης για να βρεί την βάση δεδομένων της εφαρμογής στην οποία θα γίνει το SignIn. Εάν εντοπιστεί η βάση δεδομένων, ο authentication server επιστρέφει - ανάλογα με τον client - αντίστοιχη φόρμα SignIn. Στην φόρμα θα περιέχονται τα πεδία username/email και password, με τα οποία ο client επιθυμεί να κάνει SignIn.
